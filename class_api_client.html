Detailed Description

Yksittäinen luokka backend-API-kutsujen tekemiseen.

ApiClient toimii sovelluksen keskitettynä HTTP-rajapintana. Luokka toteutetaan singleton-mallilla, jotta kaikki sovelluksen osat käyttävät samaa QNetworkAccessManager-instanssia ja samaa JWT-tokenia. Tämä helpottaa autentikointia ja vähentää päällekkäisiä yhteyksiä.
