Detailed Description

Dialogi, joka näyttää käyttäjän perustiedot JSON-datasta.

Tämä luokka vastaanottaa backendiltä saadun JSON-olion ja näyttää sen sisällön käyttöliittymässä (etunimi, sukunimi, osoite). Lisäksi dialogista voi palata takaisin tai kirjautua ulos.
